# 템플릿
- 자주 사용하는 서식을 템플릿으로 저장합니다.
- 사용자가 인식하고자 하는 텍스트 영역을 설정합니다.
    - 고정값 컬럼과 다중값 컬럼을 구분하여 저장합니다.
    - 컬럼을 표시할 순서를 저장합니다.
    - 컬럼 이름을 저장합니다.
- 예시) 전자세금계산서, 영수증, 발주서 등
